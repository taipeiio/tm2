# 說明

彙整網路戰爭規範的《[塔林手冊 2.0](https://ccdcoe.org/tallinn-manual-20-international-law-applicable-cyber-operations-be-launched.html)》在日前付梓。我們針對了書目進行了初步的翻譯，希望能進一步探討本書內容所提到的各種「網路」和「戰爭」的國際公法概念。
